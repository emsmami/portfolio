The fantasy quiz app tests your knowledge of places and people in 4 fantasy universes. The quiz app utilizes jQuery, making the landing page dynamically walk the user through a series of questions, validating responses, providing feedback on accuracy, with a final summary page judging nerdy worthiness. It was designed using common thematic elements from fantasy universes with a mobile-first concept. Each function was created to allow for precise debugging and re-utilization in any other portion of the app. Functionally, it was designed using A11y and semantic HTML to make it accesible for all users. It was a joy to create, test, and deploy. Feedback has been enthusiastic and I have been requested to make version 2.0 with an extensive library of dynamic questions for competitive purposes.

HTML5, CSS, JavaScript, jQuery

The Nature At Its Finest app combines data across 3 APIs and 4 endpoints to help the user plan trips to national parks find the local weather, and discover nearby hikes. This app utilizes jQuery, making the landing page dynamically load the search form and results from the search. It was designed using common thematic elements from PNW marketing with a streamlined clean look and mobile-first concept. Each function was created to allow for precise debugging and re-utilization in any other portion of the app. Functionally, it was designed using A11y and semantic HTML to make it accesible for all users. It was a fun challenge to pull data from one API, restructure it and send it to a separate API to get the final data request. Feedback has been very positive and requests for additional data (especially Meteor viewing) are pending responses from the API managers.
